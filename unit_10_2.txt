What is the difference between var and let?
You can't redeclare with let. Let has block scope, while var has function scope. Let is not hoisted, while var is hoisted.

What is the difference between var and const?
You can't redeclare or reassign with const. Const has block scope, while var has function scope. Const is not hoisted, while var is hoisted.

What is the difference between let and const?
Const cannot be reassigned, while let can.

What is hoisting?
The declaration of the variable is brought to the top / is first run even before it's actually assigned a value.
